Lesson 13: Understanding How CoreProtect Works

Learn how to use CoreProtect to track and undo rule violations on the VRE Minecraft SMP.

What is CoreProtect?

CoreProtect is a powerful block logging and rollback plugin for Minecraft servers. It is essential for server moderation, ensuring griefing, theft, and other rule violations can be tracked and undone. This guide will cover its core features, commands, and best practices to help moderators efficiently use the plugin.

  • Block placements & removals
  • Container interactions: Chest, furnace, etc.
  • Item transactions: Taking/placing items in containers
  • Explosions & other world interactions

CoreProtect Command Basics

Command Function
/co inspect Toggles inspection mode. Right-click blocks to see who placed or removed them.
/co rollback [time] r:[radius] u:[user] Rolls back changes made by a specific user within a given time & radius.
/co restore [time] r:[radius] u:[user] Restores blocks that were previously removed.
/co lookup [parameters] Searches logs for specific actions, users, or blocks.
/co purge [time] Clears logs older than the specified time to free up space.

Inspecting Player Actions

Using /co inspect, right-clicking a block will display:

  • Who placed or broke it
  • The time the action was performed
  • Whether an interaction occurred (e.g., a chest was accessed)

Example: Block broken by Player123 12h ago

Rolling Back Griefing & Theft

If a player griefs a build, you can roll back their actions using:

/co rollback u:Player123 t:1h r:20

  • u:Player123: Targets the user
  • t:1h: Rolls back actions from the past hour
  • r:20: Applies to a 20-block radius

Restoring Legitimate Changes

If a rollback accidentally removes legitimate changes, you can restore them:

/co restore u:Player123 t:1h r:20

This reverses a rollback and brings back removed items.

Checking Chest Transactions

To find out who took items from a chest:

/co lookup u:Player123 a:container t:12h

  • a:container: Filters results to container interactions
  • t:12h: Searches logs from the past 12 hours
